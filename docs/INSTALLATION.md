# Installation

## Prerequisites

- Node.js 20+
- npm 10+
- VS Code 1.92 or newer

## Local Setup

1. Install workspace dependencies:

```bash
npm install
```

2. Build the extension:

```bash
npm run build
```

3. Open the folder in VS Code.

4. Press `F5` to launch the Extension Development Host.

## First Run

- Open the `Chess` activity bar view.
- Run `Chess: Open Board` or `Chess: New Game`.
- The extension restores the last board state automatically if one was saved.
