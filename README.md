  <h1>Chaturanga</h1>
  <p>Play chess in VS Code — against a friend or an offline computer opponent.</p>
</div>

## Features

- Play with a friend on the same board, or against the computer (Easy / Medium / Hard)
- Classic Staunton pieces with multiple board themes
- Move history, undo / redo, board flip, and fullscreen view
- Save and resume games; import / export PGN and FEN
- Fully offline

## Usage

1. Open the **Chess** view from the Activity Bar.
2. Click **New Game** to open the board.
3. Use the sidebar to choose a mode and control the game.
